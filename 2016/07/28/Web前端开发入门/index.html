Web前端开发入门

发表于 2016-07-28   |  

Web前端，顾名思义，web系统的前端程序，狭义的也可以理解为web系统的页面开发，主要使用html，css以及javascript语言。

HTML

HTML 是一种标记语言（markup language）。它告诉浏览器如何显示内容。HTML把内容（文字，图片，语言，影片等等）和「presentation」（这个内容是如何显示，比如文字用什么颜色显示等等）分开。HTML使用预先定义的元素集合来识别内容形态。 元素包含一个以上的标记来包含或者表达内容。标记利用尖括号表示，而结束标记（用来指示内容尾端）则在前面加上斜线。

JavaScript

JavaScript是一门运行在浏览器内部的脚本语言，同时也是一门弱语言，这在导致其健壮性不强的同时也赋予了它容易学习的特点。

组成

浏览器内部的JavaScript主要由三部分组成，他们分别是ECMAScript（JS内核），DOM（文档对象模型），BOM（浏览器对象模型）。

DOM主要是指document对象，它主要负责与元素的交互
BOM对象比较多，比如location，localStorage等，它负责与浏览器相关的一些交互

作用域

JavaScript中的作用域与其他语言差别很大，只有在一个函数内部才会存在一个独立的作用域，其他大部分的结构中他们的作用域均属于父作用域。比如常见的if…else中他们的作用域就是属于副作用的。
根作用域就是全局作用域，浏览器中是window对象，nodejs中是global对象。
在该作用域下面声明的变量相对于该作用域都是全局的，那么我们的代码应该如何组织才能避免我们的变量都是在全局作用域下声明呢？
